# Campus Circle Frontend

Frontend concept for the **AI In Action** college presentation competition.

## File Structure

```text
ai in action wtih gpt/
├── index.html
├── profiles.html
├── mentor-booking.html
├── marketplace.html
├── README.md
└── assets/
    ├── css/
    │   └── styles.css
    ├── js/
    │   ├── data.js
    │   └── main.js
    └── images/
        ├── avatars/
        └── works/
```

## Pages

- `index.html`: Landing page with hero section, concept explanation, and the four featured student profiles.
- `profiles.html`: Full student directory with the four main users and extra random users.
- `mentor-booking.html`: Mentor discovery and booking layout with suggested pricing.
- `marketplace.html`: Old clothes buy/rent page with student-friendly listings.

## Notes

- The design is static frontend only.
- Sample avatars and work previews are included as SVG assets.
- Shared data is stored in `assets/js/data.js`.
